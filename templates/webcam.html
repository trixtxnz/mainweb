ℹ️ About Object Detection

Note: Object detection libraries (MediaPipe, TensorFlow.js, ml5.js) require WebGL, which is not available in Replit's workspace browser.

Solution: For object detection to work, you would need to:

  • Deploy your app and access it in a regular browser, OR
  • Use server-side detection (send frames to Python backend with OpenCV/YOLO)

This webcam demo will work here in the workspace!
